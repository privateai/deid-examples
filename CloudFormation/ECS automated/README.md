## Deployment to ECS with CloudFormation template

Private AI DEID container can be configured by CloudFormation ECS template with customer's own settings. This document illustlates how to do that. In this folder. there are 5 template yaml files for your setup.

## Files included

1. cluster.yaml
1. main.yaml
1. security-groups.yaml
1. service.yaml
1. values.yaml
